# مواعيدي - تطبيق المواعيد الذكي

## الملفات المطلوبة
```
index.html      ← التطبيق الرئيسي
sw.js           ← Service Worker (للعمل في الخلفية)
manifest.json   ← ملف PWA
icon-192.png    ← أيقونة التطبيق
icon-512.png    ← أيقونة كبيرة
```

## نشر على GitHub Pages
1. أنشئ Repository جديد على GitHub
2. ارفع جميع الملفات
3. Settings → Pages → Branch: main → Save
4. الرابط: `https://اسمك.github.io/اسم-المستودع`

## تثبيت كتطبيق على الهاتف (PWA)
### أندرويد (Chrome):
- افتح الرابط في Chrome
- قائمة (⋮) → "إضافة إلى الشاشة الرئيسية"
- سيعمل مثل تطبيق أصلي!

### iOS (Safari):
- افتح الرابط في Safari
- زر المشاركة → "إضافة إلى الشاشة الرئيسية"

## تحويل لـ APK (اختياري)
### باستخدام PWABuilder:
1. ارفع الموقع على GitHub Pages
2. اذهب لـ: pwabuilder.com
3. أدخل رابط موقعك
4. اختر Android → Download Package
5. ستحصل على APK جاهز للتنزيل
